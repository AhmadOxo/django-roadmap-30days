# Full-Stack Blog App

Live: https://djangoroadmap30.vercel.app
API: https://ahmadoxo.pythonanywhere.com/api

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Vercel
- **Backend**: Django REST Framework, JWT, PythonAnywhere
- **Auth**: JWT + Refresh Tokens

## Features
- [x] User Auth
- [x] CRUD Posts & Notes
- [x] SPA Routing
- [x] Responsive Design
